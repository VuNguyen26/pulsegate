# PulseGate v2.0.0

Status: Final release validation complete; annotated tag pending post-commit validation.

PulseGate v2.0.0 completes the fixed Product/Platform v2 roadmap across Sprints 61-80.

## Highlights

- Admin Dashboard with 18 fixed GET-only BFF resources and server-only read credentials.
- Public, static-first Developer Portal.
- Host-based and weighted upstream routing.
- Configured service discovery with bounded health-aware failover.
- Kubernetes manifests and local render/runtime documentation.
- Manual OpenTelemetry tracing foundations.
- Structured JSON logging with Loki, Grafana Alloy, and Grafana integration.
- Admin RBAC and credential-boundary hardening.
- Shared UI loading, empty, error, accessibility, and responsive polish.
- Bounded end-to-end demonstration and lightweight k6 validation.
- Final documentation, runbook, architecture, requirements, and navigation cleanup.
