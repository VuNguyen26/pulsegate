# PulseGate v1.0.0

PulseGate v1.0.0 completes Backend Portfolio v1 across Sprints 1-60.

## Highlights

- Fastify API Gateway with static and database-backed dynamic routing.
- API consumers, hashed API keys, usage plans, rate limits, and quota enforcement.
- Separate successful-usage and rejected/security analytics.
- Analytics rollup read models with bounded fallback to raw summaries.
- Guarded rollup command dry-run and direct command execute paths.
- Process-local rollup dry-run runtime behind an explicit gate.
- Non-destructive analytics retention previews and fail-closed execution review.
- Full-access and read-only Admin API key boundaries.
- Prometheus metrics, provisioned Grafana dashboard, and bounded k6 smoke validation.
- CI validation for tests, typecheck, build, and Docker image builds.

## Safety Boundaries

v1.0.0 does not add:

- Autonomous scheduled/background execute.
- External scheduler runtime execution.
- Destructive retention execution.
- Raw event deletion.
- Quota calculation from Prometheus metrics or rollup tables.
- Admin Dashboard UI or Developer Portal.
- Kubernetes, OpenTelemetry, Loki, billing, or marketplace features.

## Version Convention

`v1.0.0` is the Git and product documentation release version.

The private npm workspace package versions remain `0.1.0`; no package publication is part of this release.

The Git tag must be created only after final documentation, automated validation, runtime validation, clean repository verification, and explicit approval.
